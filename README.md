# ConvA*-PAPF

This repository contains the 3D path planning algorithm based on Differentiable A* and its application to multi-agent formation control. This work was constructed based on the inspiration from the research work of the paper "iA∗: Imperative Learning-Based A∗ Search for Path Planning".

## Project Structure

- `core/`: Core algorithm implementations.（These codes are currently being sorted out and will be available shortly）.
- `data/`: Dataset generation and loading scripts.（These codes are currently being sorted out and will be available shortly）.
- `scripts/`: Training and evaluation scripts.（These codes are currently being sorted out and will be available shortly）.
- `example/`: Application examples (e.g., 3D multi-agent formation control).
- `config/`: Configuration files for training and evaluation.
- `model/`: Pre-trained model weights (please place your `.pkl` files here).
